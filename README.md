# Customer Relationship Management

This project was generated using [Nx](https://nx.dev).

## Usage

- `yarn` - install all dependencies.
- `nx serve web-admin` - serve frontend web admin.
- `nx serve api` - serve backend api.
- `nx affected:lint` - lint changed files in the workspace.
- `nx affected:test` - test changed files in the workspace.

> In case nx is not found try `yarn run [script]`.
